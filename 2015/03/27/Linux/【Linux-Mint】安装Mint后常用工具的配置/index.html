Linux Mint

Linux Mint是一个非常年轻的操作系统，开始发行时间为2006年。不过，虽然系统本身经历的岁月不长，但由于基于成熟的软件管理，并且采用最新的Linux内涵，近年越来越受到用户追捧。
Linux Mint目标是提供一种更完整的即刻可用体验，这包括提供浏览器插件、多媒体编解码器、对DVD播放的支持、Java和其他组件。而且它与Ubuntu软件仓库兼容，它的快速发展很大程度上就是依赖Ubuntu的软件社区而流行的。

安装搜狗输入法

在搜狗输入法 for linux上下载对应你系统的deb包。
由于搜狗输入法需要fcitx作为依赖项，所以要先安装fcitx。
然后再使用sudo dpkg -i sogouinput.deb来安装搜狗输入法，安装完毕之后，注销一次，就可以使用了。

安装第三方字体

由于编程字体我比较喜欢Inconsolata字体，安装第三方字体步骤如下：

  1. cd /usr/share/fonts
  2. sudo mkdir Inconsolata
  3. 将Inconsolata-dz.otf文件复制到该文件夹中
  4. cd Inconsolata/
  5. mkfontscale
  6. mkfontdir
    这两个mkfont命令时生成Inconsolata目录下所包含的字体的索引信息
  7. fc-cache更新缓存
    然后就可以使用了

安装terminator

terminator终端可以支持一个窗口打开多个终端，可以自由的将一个终端区域横向或纵向分割建立新终端，可以进行各种个性化定制，使用起来非常方便。
安装，sudo apt-get install terminator

常用快捷键

Ctrl+Shift+E 垂直分割窗口
Ctrl+Shift+O 水平分割窗口
F11 全屏
Ctrl+Shift+C 复制
Ctrl+Shift+V 粘贴
Ctrl+Shift+N 或者 Ctrl+Tab 在分割的各窗口之间切换
Ctrl+Shift+X 将分割的某一个窗口放大至全屏使用
Ctrl+Shift+Z 从放大至全屏的某一窗口回到多窗格界面

安装ssh服务

mint下安装ssh服务使用下面的命令：

sudo apt-get install openssh-client openssh-server

通过ps -e | grep ssh来确认服务是不是已经启动。
可以通过sudo /etc/init.d/ssh start或者service ssh start来启动服务；
通过sudo /etc/init.d/ssh stop来关闭服务。
ssh-server配置文件位于/etc/ssh/sshd_config。
通过ssh username@ip_address 来登录其他机器的ssh。

截图工具shutter

shutter可以截取全屏、任意窗口、菜单和提示，功能很强大。
shutter的安装可以在软件管理器中直接搜索然后点击下载。

网页QQ

可以通过浏览器登录网页QQ，然后可以用Chromium将登录后的主页添加到桌面快捷方式。
点击More tools -> Create application shortcuts。
