No such file or directory:Python.h

我在安装IPython依赖包的时候会出现这样的错误，造成无法正确安装而退出。
出现No such file or directory这种错误，有可能是Python版本不对，有可能是真的没有Python.h这个文件。
可以进入/usr/include/python2.x文件夹下查看是否有这个文件。如果真的没有，可以安装python-dev这个包。(sudo apt-get install python-dev)

安装IPython需要的依赖包

需要下面这几种依赖包的安装：

  • pyzmq 这是用于套接字信息传递的
  • jinja2 这是一个python的模板引擎
  • tornado 这是一个web服务器软件
  • jsonschema 这是JSON Schema进行json数据格式验证的python实现

我是用包管理工具pip进行一键安装就可以了。
接下来就可以使用ipython notebook来进入web交互环境了。
