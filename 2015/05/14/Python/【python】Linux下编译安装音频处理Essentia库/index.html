Essentia在linux下的安装

Essentia(essentia github)是一个用于音频分析和基于音乐信息检索的C++开源库，它包含了一系列可复用的算法集合，像音频输入输出的工具、标准数字信号处理模块、数据的统计特征描述、一组音乐的谱、时序、音调以及更高级的描述子。另外，Gaia(gaia github)对essentia加以补充，它是一个实现了对音频分析进行相似度量和分类的C++库，并且它可以生成essentia可以使用分类模型来计算音乐的高级特征描述。

下载依赖的库

Essentia依赖以下的库：

  • FFTW: for the FFT implementation
  • libavcodec/libavformat/libavutil/libavresample (from the FFmpeg project): for loading/saving any type of audio files (optional)
  • libsamplerate: for resampling audio (optional)
  • TagLib: for reading audio metadata tags (optional)
  • LibYAML: for YAML files input/output (optional)

这里使用下面命令安装需要的库：
sudo apt-get install build-essential libyaml-dev libfftw3-dev libavcodec-dev libavformat-dev libavresample-dev python-dev libsamplerate0-dev libtag1-dev
执行这个命令之后，你可以看到它的相关依赖库是否正确配置了。

Tips：这里我遇到一个问题，这里安装的libavcodec-dev一直显示没有检测到，尽管我已经apt-get安装了这个库，后来从网上才得知，essentia需要版本是10.1以上的，而使用apt-get安装的是9.18的，于是我下载了libav的源码包再进行编译的。

另外如果使用python的绑定，python-numpy也是必须的。
对于使用python接口，需要下面的操作：
pip install sphinx pyparsing sphinxcontrib-doxylink docutils
pip install pyyaml # make sure to have libyaml installed first

编译Essentia

将以上的依赖库安装好了之后，下载源码进行配置编译。
./waf configure --mode=release --with-python --with-cpptests --with-examples --with-vamp
这个命令进行配置选项的设置。其中，–with-python to enable python bindings, –with-examples to build examples based on the library, –with-vamp to build vamp plugin wrapper。
接下来根据你的配置进行编译./waf。
然后安装C++ library和python bingdings：
sudo ./waf install。
可以进行C++和python的单元测试：
./waf run_tests和./waf run_python_tests。
